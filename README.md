# Навигационный помощник для слепых и слабовидящих
Проект представляет собой навигационного ассистента на базе компьютерного зрения и искусственного интеллекта. Система предназначена для помощи людям с нарушениями зрения в навигации, обнаружении препятствий и ориентировании в пространстве с помощью голосовых подсказок.

## Как работает система

1. Камера, микрофон и сенсоры устройства захватывают видео и аудио в реальном времени.
2. Данные передаются на сервер, где обученная модель выполняет классификацию объектов, определяет расстояние до них и положение в пространстве, а также обрабатывает голосовые команды.
3. Голосовой помощник описывает окружающую обстановку.

Примеры голосовых оповещений:
- "Справа снизу ступенька, 1 метр"
- "Спереди снизу стул, 1.5 метра"

## Функционал сервиса

- Кроссплатформенность: не важно, Android или iOS, нужен лишь браузер.
- Не требует установки приложения.
- Голосовое управление – включение камеры, фонарика и другие команды на клиенте.
- Ансамбль из YOLOv8n + дообучение YOLOv8n обеспечивает ответ за 0.1 секунды.
- Постоянное обновление данных в реальном времени.

## Запуск проекта
Для автоматического запуска необходимо иметь основной компьютер, на котором располагается сервер с моделью (сам проект), а пользователь подключается через браузер по ссылке, указанной в консоли (необходимо находиться в одной сети с сервером). Если сервер развернут удаленно и имеет SSL сертификат, то пользователю не обязательно быть подключенным к содной сети с сервером - достаточно указать адрес сервера у себя в браузере.
Чтобы развернуть автоматически окружение проекта, необходимо запустить `python3 -m run_in_venv.py` и дождаться установки всех зависимостей.

## Взаимодействие
Пользователь взаимодействует с приложением большой кнопкой "НАЧАТЬ" и голосовыми командами: "включить (выключить) камеру", "включить (выключить) фонарик".
Приложение имеет высокую масштабируемость и производительность, что позволяет расширять список доступных услуг помощника: навигация, указание прочих классификаций.

В проекте участвуют демо-версии моделей, обученые по принциау fast-forward. Для обучения полноценных расширенных моделей необходимо дообучить на больших датасетах с исходным кодом обучения авторов проекта.
